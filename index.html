🛠️ Local Installation Guide

📋 System Requirements

🐍 Python: 3.8 or higher
💾 RAM: 4GB+ recommended
💽 Storage: 2GB free space
🌐 Browser: Chrome, Firefox, Safari
🔑 API Keys: Optional (OpenAI, Google)
📦 Dependencies: Auto-installed 
# Clone the repository
git clone https://github.com/Nanimathala/hackthon.git
cd hackthon

# Create virtual environment
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # macOS/Linux

# Install dependencies
pip install -r requirements_multi_ai.txt

# Optional: Create .env file for API keys
# OPENAI_API_KEY=your_openai_key_here
# GOOGLE_API_KEY=your_google_key_here

# Run the application
streamlit run app_multi_ai.py

# Access at: http://localhost:8501

💡 Pro Tips

  • Works without API keys using built-in Fast AI mode
  • Add OpenAI/Google API keys for enhanced AI capabilities
  • Supports CPU-only operation for compatibility
  • Auto-fallback to lightweight models if needed

🔧 Troubleshooting & FAQ

❓ Common Issues

  • Port 8501 in use: Use streamlit run app_multi_ai.py --server.port 8502
  • Missing dependencies: Run pip install -r requirements_multi_ai.txt
  • API key errors: App works without keys in Fast AI mode
  • Memory issues: Close other applications, app is optimized for 4GB+ RAM

🚀 Performance Tips

  • Fast Mode: Automatically enabled for quick responses
  • PDF Size: Keep PDFs under 50MB for optimal processing
  • Browser: Use Chrome or Firefox for best experience
  • Network: Stable internet needed for OpenAI/Gemini APIs

Need more help? Check the GitHub repository for detailed documentation and issue tracking.
